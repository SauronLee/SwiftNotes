    データ・タイプ¶

    定数と変数¶

    let 定数はコンパイル時に決定する必要はありませんが、使用する前に一度だけ代入する必要があります。
    1var num = 10
2num += 20
3let age = num

    1func getAge() -> Int {
2    return 10
3}
4let age = getAge()

    定数や変数は、初期化されるまでは使用できません。

    1let age: Int
2var height: Int
3print(age) //error
4print(height) //error

    Optional (?)¶

    allowable nil

    1var name: String? = "Jack"
2name = nil
3
4var age: Int? //The default value is nil
5age = 10
6age = nil

     1var array = [1, 15, 40, 29]
 2func get(_ index, Int) -> Int? {
 3    if index < 0 || index >= array.count {
 4        return nil
 5    }
 6    return array[index]
 7}
 8print(get(1)) // optional(15)
 9print(get(-1)) // nil
10print(get(4)) // nil

    Nil-Coalescing Operator (??)¶

    A ?? B as Operator ?? Whatever if A is nil return A if A is not nil return B, if B is not Operator return A!

    1let a: Int? = 1
2let b: Int? = 2
3let c = a ?? b // c.type is Int?, Optional(1)

    1let a: Int? = nil
2let b: Int? = 2
3let c = a ?? b // c.type is Int?, Optional(2)

    1let a: Int? = nil
2let b: Int? = nil
3let c = a ?? b // c.type is Int?, nil

    1let a: Int? = 1
2let b: Int = 2
3let c = a ?? b // c.type is Int, 1

    1let a: Int? = nil
2let b: Int = 2
3let c = a ?? b // c.type is Int, 2

    Source code

    1public func ?? <T>(optional: T?, defaultValue: @autoclosure () throws -> T?) rethrows -> T?
2public func ?? <T>(optional: T?, defaultValue: @autoclosure () throws -> T) rethrows -> T

    Implicitly Unwrapped Optional¶

    変数 Int! が常に値を持っていることを確認する必要があります。

    1let num1: Int! = 10
2let num2: Int = num1
3if num1 != nil {
4    print(num1 + 6) // 16
5}
6if let num3 = num1 {
7    print(num3)
8}

    1let num1: Int! = nil // Fatal error
2let num2: Int = num1

    String Interpolation¶

    1var age: Int? = 10
2print("My age is \(age)") // error!!!
3// rewrite
4print("My age is \(age!)") // My age is 10
5print("My age is \(String(discribing: age))") // My age is Optional(10)
6print("My age is \(age ?? 0)") // My age is 10
